Jak to publikovat

Žádný black box — pracujete s reálnými soubory webu.

1. Upravte stránku (texty, obrázky, bloky, menu). Změny se průběžně ukládají do prohlížeče jako koncept.

2. Klikněte na Stáhnout HTML — stáhne se soubor index.html dané stránky.

3. Nahrajte ho na hosting do správné složky (např. /sluzby/social-bot/index.html), nebo commitněte do GitHub repozitáře — web se sám aktualizuje.

Nová stránka: po stažení ji nahrajte do složky /vas-slug/index.html. Pokud jste ji přidali do menu, stáhněte i ostatní stránky, aby měly nové menu.

Tip: Pro plně automatické publikování (jeden klik) stačí napojit web na Netlify/Vercel + jednoduchý ukládací endpoint — řekněte si o doplnění.
